# TORAM
TORAM is a language designed to be easy to compile to [≈Åukasz Szkup's version](https://www.szkup.com/?pid=praca_mgr) of a [random-access machine (RAM)](https://en.wikipedia.org/wiki/Random-access_machine), while having functions and named variables.

## USAGE

`python3 run.py path/to/file.TORAM` to compile and interpret TORAM code
`python3 run.py -c path/to/file.TORAM` to compile TORAM code and print the result (in RAM's language)
